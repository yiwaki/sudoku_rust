# sudoku_rust

Python module to solve Sudoku puzzle. <https://en.wikipedia.org/wiki/Sudoku>

## Install Maturin
```
pip install maturin
```

## Download
```
git clone https://github.com/yiwaki/sudoku_rust.git
```

## Compile and install
```
cd sudoku_rust
make develop
```

## run
```
python test.py
```
