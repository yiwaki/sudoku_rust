# sudoku_rust

Program to solve Sudoku puzzle <https://en.wikipedia.org/wiki/Sudoku>

1. Install Maturin
```
pip install maturin
```

2. Download
```
git clone https://github.com/yiwaki/sudoku_rust.git
```

3. Compile and install
```
cd sudoku_rust
make develop
```

4. run
```
python test.py
```

5. how to make Sudoku puzzle

see sample data -> data/easy.csv
