# sudoku_rust

A program to solve Sudoku made with Rust (python module).

For more information on Sudoku, see [Wikipedia](https://en.wikipedia.org/wiki/Sudoku).

## Install Maturin
```
pip install maturin
```

## Download
```
git clone https://github.com/yiwaki/sudoku_rust.git
```

## Compile and install
```
cd sudoku_rust
make develop
```

## Run sample program
```
python test.py
```
